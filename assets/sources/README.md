## assets/sources

Este diretório é apenas um placeholder para manter o caminho `assets/sources/`
estável no repositório.

As *sources* reais do KodexB vivem no diretório configurado pelo usuário
(`--source` / config), tipicamente `~/.config/kodexb/sources/`, e **não** devem
ser commitadas aqui.
